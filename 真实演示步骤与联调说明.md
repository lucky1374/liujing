## 10. 常见问题排查

### 10.1 前端请求失败

检查：

- 后端是否启动
- `http://localhost:3000/api-docs` 是否可访问

### 10.2 MySQL 连接失败

检查：

- `.env` 中数据库密码是否正确
- MySQL 数据库是否存在

### 10.3 Python Runner 执行失败

检查：

- `python-runner/.venv` 是否已创建
- `pip install -r requirements.txt` 是否执行
- 后端是否已重启以识别 `.venv/bin/python`
